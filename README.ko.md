# deep-evolve

Claude Code용 자율 실험 플러그인. 목표를 지정하면 deep-evolve가 측정 기반 실험 루프를 통해 프로젝트를 체계적으로 개선합니다.

## 작동 방식

1. `/deep-evolve` — 프로젝트를 분석하고 평가 하네스를 생성합니다
2. 실험 루프 실행: 코드 수정 → 평가 → 개선 사항 유지, 회귀 폐기
3. 수확 체감 분석과 함께 결과를 보고합니다

## 빠른 시작

```bash
# Claude Code에서 실행:
/deep-evolve

# 특정 횟수를 지정하여 실행:
/deep-evolve 50
```

## 방법론

세 가지 도메인에서 검증됨:
- **ML 학습**: val_bpb 최적화 (100회 이상 실험)
- **퀀트 트레이딩**: 복합 점수 최적화 (100회 실험에서 +12.3%)
- **플러그인 엔지니어링**: 10개 버그 발견 및 수정 (60회 실험, 146개 테스트 시나리오)

## 지원 도메인

| 도메인 | 평가기 | 예시 |
|--------|--------|------|
| ML / 학습 | stdout 메트릭 파싱 | val_bpb, loss, accuracy |
| 테스팅 | 테스트 통과율 + 커버리지 | jest, pytest, cargo test |
| 코드 품질 | 커스텀 시나리오 | 보안 패턴, 린트 규칙 |
| 전략 | 백테스트 결과 | 샤프 비율, 최대 낙폭 |

## 설치

```bash
claude plugin add /path/to/deep-evolve
```
